# 🌿 Explore Passa Quatro

Site turístico desenvolvido como projeto de extensão universitária com o objetivo de divulgar pontos de interesse da cidade de Passa Quatro - MG.

## 📌 Sobre o projeto

O site apresenta sugestões de locais para visitação na cidade, organizados em categorias para facilitar a experiência do usuário:

- 🌿 Natureza  
- ⛰️ Aventura  
- 👨‍👩‍👧 Família  
- 🍽️ Gastronomia  

Cada local possui descrição, imagens, referência de localização e espaço para avaliações de visitantes.

## 💬 Sistema de avaliações

O site conta com um sistema simples de comentários, onde os usuários podem deixar:
- Nome
- Comentário

Sem necessidade de login.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 🎯 Objetivo

Promover o turismo local e facilitar o acesso a informações sobre pontos turísticos, incentivando a visitação e valorização da cidade de Passa Quatro - MG.
